# Neural Network Learning Platform

An interactive educational platform built from scratch to understand how neural networks learn through experimentation and visualization.

## 🚀 Features

- Perceptron and Multi-Layer Perceptron (MLP)
- Forward propagation and backpropagation
- Gradient descent optimization
- Interactive configuration (architecture, learning rate, etc.)
- Visualization:
  - Loss vs Epochs
  - Accuracy vs Epochs
  - Decision boundary
  - Confusion matrix
- Experiment tracking
- Comparison with sklearn models

## 🧠 Key Concepts Demonstrated

- Linear vs non-linear classification
- Effect of network depth
- Overfitting and regularization
- Bias vs variance tradeoff

## 🧪 Experiments

- Perceptron vs MLP
- Effect of hidden layers
- Regularization techniques

## 🛠 Tech Stack

- Python
- NumPy
- Matplotlib
- scikit-learn
- ipywidgets (interactive UI)
